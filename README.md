# RipperFoxExtensios
The Firefox add-on part of RipperFox ported to Chrome for your chrome needs, choom.

## The backend can be found here
[RipperFox](https://github.com/Yama-K/RipperFox), or if you just want the backend [click me](https://github.com/Yama-K/RipperFox/releases/download/Release/RipperFox.exe)

## Install 
1. Download the release -> [Download](https://github.com/Yama-K/RipperFox-ChromeExtension/releases/download/Release/RipperFox-ChromeExtension.zip)
2. Unpack it somewhere
3. Go to chrome://extensions
4. Enable developer mode (Top right corner)
5. Choose "load unpacked"
6. Select your unpacked folder
7. Done
